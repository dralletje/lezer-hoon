# Lezer grammar for Hoon

This is a [Hoon](https://developers.urbit.org/reference/hoon) grammar for the [lezer](Lezer.codemirror.net/) parser system.

## Contributing

Initializing dev environment:

```
npm install
make
```

Running tests:

```
make test
```
